使用`toJSON`即可获取我们需要的格式

```javascript
let obj = await this.ctx.model.User.find({ where: { name: name } });
console.log(obj.toJSON());
```

打印如下(我们需要的数据)

```json
{ id: 2,
  name: 'test',
  age: 18,
  created_at: 2019-04-02T03:50:33.000Z,
  updated_at: 2019-04-02T03:50:33.000Z }
```
